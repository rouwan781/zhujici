# 🎯 助记词扫描器

一个用于教育目的的比特币助记词扫描工具。

## 功能特点
- 🔍 实时生成和检查助记词
- 📊 完整的统计信息
- 🔔 余额发现警报
- 📱 响应式设计
- 💾 数据导出功能

## 使用说明
1. 点击开始扫描按钮开始
2. 在第1000次和第10000次扫描时会显示预设地址
3. 发现余额时会有警报提示
4. 可以使用导出功能保存数据

## 注意事项
⚠️ 仅供教育用途
- 生成有效助记词概率极低
- 不存储任何用户数据
- 所有操作在浏览器本地完成

## 技术栈
- HTML5  CSS3  JavaScript
- BIP39 标准
- 区块链API
